# spring-boot-foundation

This repository is a hands-on learning and exploration project for Spring Boot,
focused on understanding its core concepts, runtime model, and production-oriented practices.

The goal is not to build a demo application, but to establish a solid foundation
for long-running, maintainable Java backend services using Spring Boot.
